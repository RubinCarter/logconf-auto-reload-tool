### log4j(slf4j) 日志动态升降级实现方案   

**使用说明** 
1. 请参考 com.chris.log.listener.MyLog4jConfigListener 类文件中的使用说明  

 **技术选型：**  
   
	核心框架：Spring Framework 4.2   
	视图框架：Spring MVC 4.2   
	安全权限框架：Apache Shiro 1.3   
	持久层框架：MyBatis 3.3   
	定时器：Quartz 2.2   
	数据库连接池：Druid 1.0   
	日志管理：slf4j 1.7.x、log4j 1.2.x 
	页面交互：Vue2.x   
   

 **软件需求** 
    
	JDK1.7+ 
	MySQL5.5+     
	Tomcat7.0+     
	Maven3.0+
